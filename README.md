# PowershellKerberos
Some scripts to abuse kerberos using Powershell

## Injector.ps1
Small tool for injecting kerberos tickets. Supports two work modes:
- U can read ticket from kirbi file (1 mode)
- U can read ticket from b64 (2 mode)
